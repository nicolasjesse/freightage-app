# Sistema de Fretagem

## Descrição
O sistema visa possibilitar um gerenciamento para companhias de fretagem e o controle de seus serviços
## Pré requisitos
* Ruby 3.0+
* Rails 7.0+

## Como rodar
### Servidor
```
$ bundle install
$ rails db:create
$ rails s
```
### Testes
```
$ rspec
```
